# ЛР0 — Решение уравнения шестой степени

Программа находит вещественные корни уравнения вида:

```
a·x⁶ + b·x⁵ + c·x⁴ + d·x³ + n·x² + m·x + k = 0
```

Коэффициенты вводятся пользователем. Программа определяет фактическую степень многочлена и подбирает подходящий метод решения, после чего понижает степень и повторяет процесс.

## Применённые методы

| Степень | Метод |
|---------|-------|
| ≥ 6 | Метод касательных (Ньютона) |
| 5 | Метод хорд (секущих) |
| 4 | Комбинированный метод (хорды + касательные) |
| 3 | Формула Кардано |
| 2 | Формула дискриминанта (Виета) |
| 1 | Прямое вычисление |

Понижение степени после нахождения каждого корня — по **схеме Горнера** (деление на `(x − root)`).

## Запуск

```bash
python main.py
```

Программа запросит коэффициенты `a, b, c, d, n, m, k`, а для численных методов — параметры (погрешность, число итераций, границы поиска) либо значения по умолчанию.

## Пример

```
Введите a: 0
Введите b: 0
Введите c: 0
Введите d: 1
Введите n: -6
Введите m: 11
Введите k: -6

ИТОГОВЫЕ КОРНИ:
x1 = 1.000
x2 = 2.000
x3 = 3.000
```
